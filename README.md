# Car Rental Fullstack Project

This is a premium Car Rental application built with React (Vite) and Node.js (Express).

## Project Structure

- `client/`: React frontend
- `server/`: Express backend

## Deployment

This project is configured for deployment on Vercel. 
- The `client/` directory contains its own `vercel.json` for frontend routing.
- The `server/` directory contains its own `vercel.json` for API serverless functions.
